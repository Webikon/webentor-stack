Core Blocks Catalog ​

  • e-accordion: Single accordion item with title and content.
  • e-accordion-group: Container for e-accordion items.
  • e-breadcrumbs: Renders breadcrumbs (filter path via webentor/breadcrumbs/path).
  • e-button: Button with variants/sizes and link.
  • e-icon-picker: Icon rendering with color and link options.
  • e-image: Image block with lightbox/link and responsive size controls.
  • e-picker-query-loop: Query loop by picked posts.
  • e-post-template: Inner template for query loop items.
  • e-query-loop: Dynamic query loop (per-page, tax filters, etc.).
  • e-slider: Slider container for arbitrary inner blocks.
  • e-svg: Raw SVG renderer.
  • e-tab-container: Single tab (title + content), used in tabs.
  • e-tabs: Tabs container.
  • e-table: Table container.
  • e-table-row: Table row.
  • e-table-cell: Table cell (th/td) with span options.
  • l-404: 404 layout section.
  • l-flexible-container: Flex container.
  • l-footer: Footer layout.
  • l-formatted-content: Rich content wrapper.
  • l-header: Header layout.
  • l-mobile-nav: Mobile navigation.
  • l-nav-menu: Navigation menu.
  • l-post-card: Post card template.
  • l-section: Section wrapper.
  • l-site-logo: Site logo.
